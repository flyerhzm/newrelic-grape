# NewRelic::Grape

NewRelic instrument for [grape][0], this is inspired from this [blog post][1].

## Installation

Add this line to your application's Gemfile:

    gem 'newrelic-grape'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install newrelic-grape
